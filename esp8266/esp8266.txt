
I've included the latest files as of 2017-10-02 but please download the latest versions from the sources!

Download firmware here: (esp8266_deauther_512kb.bin)
https://github.com/spacehuhn/esp8266_deauther/releases

Download NodeMCU Flasher here:
https://github.com/nodemcu/nodemcu-flasher

Open Device Manager and look for "Silicon Labs CP210x USB to UART Bridge" (devmgmt.msc)
Check which COM port it's connected to. Should be displayed at the end of the name.

Open NodeMCU flasher.

Set COM Port to whatever your NodeMCU has.
Select the .bin file you downloaded in the Config tab.

In Advanced, set settings as following:
Baudrate: 230400
Flash size: 4Mbyte
Flash speed: 80MHz
SPI Mode: DIO

Go back and click to Operation and click flash. Hopefully you're all set! (Takes about 1 minute.)
Disconnect when you see a green checkmark.
